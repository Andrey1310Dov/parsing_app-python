Description
===========

Данное приложение работает с указанным log-файлом (log-файлы добавляются в специальную папку,  
а далее по указанию пользователя конкретный файл считывается) и выводит следующую информацию:

- С какого ящика и сколько писем было отправлено.
- Сколько писем отправлено успешно и сколько с ошибками.

Данные подсчитываются только по вновь зарегристрированным пользователям.
